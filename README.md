## explane [![justforfunnoreally.dev badge](https://img.shields.io/badge/justforfunnoreally-dev-000)](https://justforfunnoreally.dev)
a small popup that explains any text you highlight on your browser.

## before you run
- make sure you add your API KEY, API URL and the model you need to the *background.js* file
- make sure you add your API URL to the *manifest.json* file

## installation
- clone this repo or download the files (background.js, content.js, manifest.json, style.css) in a folder.
- open your browser's extension settings, check if there exists any developer setting mode or option to upload custom extenstions. 
- upload the manifest.json file or a zip of the files (requirement different per browser).

## reminder
- this extension doesn't ask for any permissions other than the AI APIs for your account or reading the tab data. if it does ask, make sure you have installed the correct files or cloned the correct repo.
- **any loss of data or private information is your responsibility. this is a basic and simple project that doesn't take or give private data to anyone or anwyhere. any selected text is sent to the AI of your selection, so be aware of what you are highlighting.**

## example usage
highlight any text you want, a small black square with a white '?' will apprear on the bottom left, click that to have your model display the information about the highlighted text at the bottom right of the screen.
